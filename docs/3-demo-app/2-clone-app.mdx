# üëê Clone the application

Go to the [Github repository](https://github.com/mongodb-developer/docs-vector-search-demo) for the demo application. 

First, clone the repository to your local machine.

```bash
git clone git@github.com:mongodb-developer/docs-vector-search-demo.git
```

Then, change to the `docs-vector-search-demo` directory.

```bash
cd docs-vector-search-demo
```

Install the dependencies.

```bash
npm install
```
